Aplicación SocialLife

Información sobre la app

SocialLife es una aplicación de realidad aumentada que busca concientizar a las personas a través de actividades de labor social. La aplicación permite que los usuarios generen misiones que deben ser realizadas por otros usuarios, trabajando juntos hacia una comunidad más próspera. La aplicación cuenta con una tienda de recompensas donde los usuarios pueden gastar las SocialCoins adquiridas al completar misiones.

Beneficios de usar SocialLife

Denuncias

La aplicación SocialLife permite registrar denuncias de los usuarios a través de la captura de evidencia durante la realización o creación de misiones específicas.

Informar Emergencias

Podremos contactar con emergencias y solicitar ayuda, lo que enviará un aviso al servicio de emergencias con nuestra posición exacta.

Acciones Sociales

La aplicación puede ser utilizada para incentivar a las personas a realizar acciones sociales para la comunidad teniendo presente la solidaridad

Objetivos de la app

- Promover la labor social de la gente para así tener una ciudad limpia y segura.

- Recompensar y motivar las buenas acciones de las personas.

Como funciona el manejo de SocialCoins (Moneda Interna Del Juego):

Las monedas internas del juego (SocialCoins) funcionan como la ganancia que el usuario obtiene al finalizar una misión o la creación de una labor social.

Estas son las siguientes formas para obtener SocialCoins:

-Horas de juego: Es necesario dedicar una cantidad promedio (1 hora diaria) para ir recolectando las suficientes SocialCoins por medio de las misiones normales para que se vayan acumulando y así tener una recompensa.

-Inicio de sesión diario: Al iniciar sesión diariamente se les otorgara una cantidad limitada de SocialCoins de manera gratuita por iniciar sesión y estar al pendiente al día con día de la utilización de la aplicación.

-Anuncios: Habrá apartado de anuncios para que el usuario desea recibir SocialCoin de manera rápida, este apartado tendrá un límite diario de 3 anuncios diarios

Funciones que incluyen la aplicación

Las funciones principales de la aplicación son las siguientes:

-Introducir una misión: El usuario podrá introducir misiones utilizando el apartado de “introducción de misión”. Los usuarios pueden resolver la misión de manera satisfactoria cumpliendo los requisitos de esta.

-Realizar misiones: el usuario puede realizar misiones y tomar fotografías como evidencia del cumplimiento. En ciertos casos, la evidencia obtenida puede servir para que los organismos públicos atiendan las denuncias. Al completar misiones, el usuario recibirá SocialCoins y, mientras más misiones se completen, más SocialCoins se obtendrán.

-Canjear SocialCoins:El usuario podrá canjear las SocialCoins que ha ido ganando cumpliendo misiones, las SocialCoins sirve para obtener las recompensas que están disponibles en la tienda, cada producto tiene un valor de SocialCoins.
